Capabilities

Professional-Grade Security Tools

AES-256 Encryption

Operations

6 Operating Modes

Flexible operation modes for every stage of your security assessment workflow.

01

Monitor

Full-screen live dashboard with real-time keystroke display, stats, and resource meters.

python main.py --mode monitor
02

Stealth

Background monitoring without any visible UI. Logs silently to encrypted files.

python main.py --mode stealth
03

Review

Browse and inspect collected log files with metadata, sizes, and encryption status.

python main.py --mode review
04

System Info

Comprehensive system profiling: OS, hardware, network interfaces, and running processes.

python main.py --mode sysinfo
05

Decrypt

Decrypt encrypted logs and export to TXT, CSV, or JSON for analysis.

python main.py --mode decrypt
06

Test

Run comprehensive self-diagnostic unit tests to verify all modules are functioning.

python main.py --mode test

Responsibility

Built with Ethics First

CyberSentinel enforces ethical usage at every level. These aren't just guidelines — they're built into the code.

✅

Consent Enforcement

Interactive consent prompt with full legal terms must be accepted before any monitoring begins. No bypass possible.

📝

Audit Trail

Every consent decision is logged with timestamps and machine-hash identification. Full accountability.

⏰

Auto-Stop Timer

Configurable safety timer automatically stops monitoring after a set duration (default: 8 hours).

🔒

Encrypted Storage

All captured data is encrypted at rest using industry-standard Fernet encryption. Keys rotate automatically.

🏷️

Session Watermarks

Every log file contains session metadata watermarks for identification and traceability.

🚫

Email Disabled

Email reporting is disabled by default. Must be explicitly configured with environment variables.
